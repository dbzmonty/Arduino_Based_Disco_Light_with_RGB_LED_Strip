Arduino Based Disco Light with RGB LED Strip
--------------------------------------------

Created by Andras Dobozi
Eszterhazy Karoly University
2017

----------------------------------------------------------------

This is an Atmel microcontroller based RGB LED strip controller.
The microphone is sensing the beat of the music and sends signal
to the microcontroller to change the colour of the strip.

----------------------------------------------------------------

Required parts:
---------------

Atmel ATMega8 or ATMega328P microcontroller + FTDI programmer
7805 5V voltage regulator
2x 100uF capacitor
2x 22pF capacitor
16MHz crystal
10K resistor
3x 220 Ohm resistor
3x BD139 NPN transistor
12V 2A DC power supply
max. 5m RGB LED strip
Microphone Sensor for Arduino
